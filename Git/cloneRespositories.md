# Clone respositories

## Mac (ver.2022/10/14)
### step 01/ 取得ssh金鑰
1. `$ ssh-keygen`
2. follow steps
3. get key

### step 02/ github 連動
1. account > setting > SSH and GPG keys
2. New SSH key
3. name a title, paste on the key

### step 03/ clone repositories
1. open the repositories
2. copy url  
   clone > https //readOnly  
   clone > SSH  //read and Write
3. Back to terminal  
```
$ cd #directory
$ git clone #url
```
